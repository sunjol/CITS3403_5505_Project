Educational Flask app

MVC and Databases

A tiny web app showing MVC, SQLite, SQLAlchemy ORM, CRUD, relationships, controllers, templates and front-end API calls.

What this app demonstrates

Model

app/models.py defines Student and ProjectGroup.

View

templates/index.html and static/css/styles.css render the page.

Controller

routes.py receives requests and controllers.py performs app logic.

Database

SQLite stores rows. SQLAlchemy maps rows to Python objects.

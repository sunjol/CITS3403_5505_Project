Why this rewrite is useful

Bootstrap reduces the amount of CSS you need to write by providing reusable classes for spacing, layout, buttons, forms, and more.

A good way to study Bootstrap is to compare each section here with the original version and notice which custom CSS rules are no longer needed.

Mini Bootstrap checklist

Use .container for page width
Use .row and .col-* for layout
Use .card for boxed sections
Use .form-control and .form-select for forms
Use utility classes for spacing and colours

Small improvement over the original

The page is now more responsive, cleaner on mobile, and easier to extend. That makes it better for learning modern front-end structure.

# User Stories – PromptShare Website

## 1. Visitor and Account Access

1. **As a visitor, I want to view the homepage so that I can understand what PromptShare does and why it is useful.**

2. **As a visitor, I want to sign up for an account so that I can access prompt optimisation and saving features.**

3. **As a returning user, I want to log in securely so that I can access my dashboard, prompts and history.**

4. **As a logged-in user, I want to log out so that I can protect my account when I finish using the website.**

---

## 2. Dashboard and User Overview

5. **As a user, I want to view a dashboard so that I can see an overview of my prompts, recent activity and useful shortcuts.**

6. **As a user, I want to see my daily prompt polishing quota on the dashboard so that I know how many optimisation requests I have left today.**

7. **As a user, I want my daily quota to reset automatically at midnight so that I receive a fresh allowance each day.**

---

## 3. Prompt Creation and Management

8. **As a user, I want to create a new prompt so that I can save, optimise or test it later.**

9. **As a user, I want to view a list of my saved prompts so that I can browse, manage and reuse my work.**

10. **As a user, I want to view the details of a specific prompt so that I can see its content, versions, optimisation results and history.**

11. **As a user, I want to edit an existing prompt so that I can improve it over time.**

12. **As a user, I want to delete prompts I no longer need so that my saved prompt list stays organised.**

---

## 4. Prompt Versions and History

13. **As a user, I want to save multiple versions of a prompt so that I can track how it changes over time.**

14. **As a user, I want to view the version history of a prompt so that I can compare earlier and newer versions.**

15. **As a user, I want to compare an original prompt with its optimised version so that I can understand what was improved.**

---

## 5. Prompt Optimisation and AI Testing

16. **As a user, I want to submit a prompt for optimisation so that my basic prompt can be rewritten into a clearer and more structured version.**

17. **As a user, I want to test a prompt with an AI model so that I can see the generated output.**

18. **As a user, I want optimisation results to be saved automatically so that I can revisit them later.**

19. **As a user, I want to receive a clear message when a prompt cannot be optimised so that I understand what went wrong.**

20. **As a user, I want the system to prevent optimisation when my daily quota is exhausted so that quota limits are applied fairly.**

---

## 6. Search and Filtering

21. **As a user, I want to search my prompts by keyword so that I can quickly find specific prompts I have created or saved.**

22. **As a user, I want to filter prompts by visibility, date or usage status so that I can manage my prompt collection more easily.**

---

## 7. Public Sharing and Community

23. **As a user, I want to mark a prompt as public or private so that I can control whether other users can see it.**

24. **As a user, I want to browse public prompts shared by other users so that I can learn from useful examples.**

25. **As a user, I want to view details of a public prompt so that I can understand how another user structured it.**

26. **As a user, I want to upvote useful public prompts so that high-quality prompts are highlighted in the community.**

27. **As a user, I want to search public prompts by keyword so that I can find relevant community examples.**

---

## 8. Profile and Account Management

28. **As a user, I want to view my profile so that I can see my account details and prompt activity.**

29. **As a user, I want to update my profile information so that my account details stay accurate.**

30. **As a user, I want my private prompts and account data to be protected so that other users cannot access or modify them without permission.**

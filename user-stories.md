User Stories- Prompt Share Website
As a user, I want to create an account so that I can save and manage my prompts.
As a user, I want to log in securely so that I can access my personal dashboard.
As a user, I want to create a new prompt so that I can experiment with AI responses.
As a user, I want to edit an existing prompt so that I can improve its quality over time.
As a user, I want to save multiple versions of a prompt so that I can track changes and improvements.
As a user, I want to test my prompt with an AI model so that I can see the generated output instantly.
As a user, I want to view a history of prompt versions so that I can compare past and current results.
As a user, I want to make my prompts public or private so that I can control who can see my work.
As a user, I want to browse prompts shared by other users so that I can learn from and reuse effective prompts.
As a user, I want to upvote useful prompts so that high-quality prompts are highlighted in the community.
As a user, I want to search for prompts using keywords so that I can quickly find relevant content.
As a user, I want to use a prompt optimizer so that my basic prompts can be improved into structured formats.
